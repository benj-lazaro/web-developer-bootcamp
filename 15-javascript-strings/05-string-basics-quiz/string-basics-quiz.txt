String Basics Quiz
------------------
- Given the following string:
    
    const animal = "hippopotamus"

    - Which character am I accessing? animal[7]
    - Answer: "t"


- What is stored in the combo variable?

    const city = 'Kyoto';
    const country = 'Japan';
    const combo = city + country;

    - Answer: "KyotoJapan" 


- What does the following return?

    let year = "1998";
    year + 1;

    - Answer: "19981"
