flex-direction
--------------
- To enable Flexbox in CSS
    - On the element designated as a container (flex container)
        - Declare the CSS property "display"
        - Assign the value of "flex"

    - By default
        - Lines up elements (items) in a container from left to right
        - When the CSS property "flex-direction" is / has NOT yet been declared


- Inside a flex container
    - There are two (2) axis
        - Main axis
        - Cross axis
    
    - Main axis determines how the items flow inside a container


- The CSS property "flex-direction" sets the flex container's Main axis
    - As well as the flow of the container's items 

    - "row" (default)
        - Items inside a container flows from left to right

    - "row-reverse"
        - Items flow from right to left

    - "column"
        - Items flow from top to bottom
    
    - "column-reverse"
        - Items flow from bottom to top

    - Reference (https://developer.mozilla.org/en-US/docs/Web/CSS/flex-direction)
