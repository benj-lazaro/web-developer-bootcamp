innerHTML, textContent & innerText
----------------------------------
- The "Element" object property ".innerHTML"
    - It gets the full content of the selected "Element" object
        - Including the embedded HTML elements (or tag names)

    - It sets (updates) the content of the selected "Element" object
        - Capable of "understanding" HTML element(s) embedded in the given value

    - IMPORTANT: It accepts & parses argument value as HTML
        - Writing the results into the DOM
        - Offers a POTENTIAL vector for Cross-Side Scripting (XSS)

    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/Element/innerHTML)


- The "Node" object property ".textContent"
    - It is similar to the property ".innerText" EXCEPT unlike ".innerText"
        - Every embedded HTML mark-up is replaced w/ whitespace

    - It gets the every piece of content inside the selected "Element" object
        - Including hidden content & line breaks

    - IMPORTANT: Setting the property ".textContent"
        - REMOVES ALL pre-existing "Element" objects & content
        - REPLACES it them w/ the given (string) value

    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/Node/textContent)


- The "HTMLElement" object property ".innerText"
    - It gets the human-readable content of the "Element" object
        - IGNORES HTML elements (tag names) embedded in the content

    - It sets the text content of the selected "Element" object w/ the given value
        - It accepts the passed value as a string
        - Embedded HTML element(s) w/in the passed value will be treated as text
    
    - IMPORTANT: Setting the property ".innerText"
        - REMOVES ALL pre-existing "Element" objects & content
        - REPLACES it them w/ the given (string) value

    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/HTMLElement/innerText)
