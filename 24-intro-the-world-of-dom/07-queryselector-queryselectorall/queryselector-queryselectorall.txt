querySelector & querySelectorAll
--------------------------------
- The "document" object method ".querySelector()"
    - Better (& newer) way to select an "Element" object from a web page
    
    - Acceptable argument values are as follows
        - A HTML tag
        - A HTML attribute
        - Any CSS style selector

    - Returns the 1st matching "Element" object
        - Otherwise, returns "null"

    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/Document/querySelector)


Syntax:
document.querySelector(<html_tag_or_css_selector>);


- The "document" object method ".querySelectorAll()"
    - Better (& newer) way to select ALL matching "Element" objects from a web page

    - Acceptable argument values are as follows
        - A HTML tag
        - A HTML attribute
        - Any CSS style selector

    - Returns a NodeList of matching "Element" objects
        - Otherwise, returns an empty NodeList

    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/Document/querySelectorAll)

    - IMPORTANT: A NodeList is an itereable array-like object that contains matching "Element" objects
        - Unlike a HTMLCollection, it is NOT LIVE
        - Requires to call the method "document.querySelectorAll()" AGAIN
        - In order to stay in-sync w/ the DOM tree


Syntax:
document.querySelectorAll(<html_tag_or_css_selector>);
