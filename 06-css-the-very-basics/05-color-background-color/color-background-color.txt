color And background-color Properties
-------------------------------------
- The CSS property "color"
    - Sets the foreground color value of an HTML element's text & text decorations

    - It accepts various types of value (i.e. color system)
        - Named colors, Hexadecimal, RGB, RGBA, HSL, HSLA
        - Each value represent a color

    - Reference (https://developer.mozilla.org/en-US/docs/Web/CSS/color)


- The CSS property "background-color"
    - Sets the background color value of an HTML element

    - It accepts various types of value (i.e. color system)
        - Named colors, Hexadecimal, RGB, RGBA, HSL, HSLA
        - Each value represent a color

    - Reference (https://developer.mozilla.org/en-US/docs/Web/CSS/background-color)

    - The CSS property "background" offers similar behavior EXCEPT it more powerful
        - It is a shorthand for the following CSS properties
            - "background-color"
            - "background-image"
            - "background-origin"
            - "background-position"
            - "background-repeat"
            - and more

        - Reference (https://developer.mozilla.org/en-US/docs/Web/CSS/background)
