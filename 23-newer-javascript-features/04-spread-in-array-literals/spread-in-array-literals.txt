Spread In Array Literals
------------------------
- The spread syntax (...) 
    - It allows an iterable to be expanded in places where
        - 0 or more elements (array) are expected
        
    - The elements of the arrays that are passed as elements
        - Using the spread syntax remains UNCHANGED

    - It offers as a convenient alternative to the method ".slice()"

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Spread_syntax)


Syntax:
const <array> = [...<iterable>, <element_1>, ..., <element_n>];
