Destructuring Params
--------------------
- Destructuring a function's parameter list
    - On a function's parameter list
        - Specify specific properties to be extracted from the passed object
        - Enclose properties w/in a pair of "{ }"
        - Return the extracted properties

        - NOTE: Frequently used on objects passed as argument values

    - To assign a default value to a non-existent property
        - Declare the (non-existent) property in a function's parameter list
        - Use the "=" operator
        - Followed by the default value


Syntax:
const <name> = ({<property_name>, ..., <property_name>}) => {
    ...
    return `${<property_name>} ... ${<property_name>}`;
};

const <name> = ({<non-existent_property_name> = <default_value>}) => {
    ...
    return `${<property_name>} ... ${<property_name>}`;
};
