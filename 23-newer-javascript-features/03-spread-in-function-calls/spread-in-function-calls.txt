Spread In Function Calls
------------------------
- The spread syntax (...) 
    - It is used on multiple situations
        - Like a "swiss army knife" in JavaScript

    - It allows an iterable to be expanded in places where
        - 0 or more arguments (function) or elements (array) are expected
        
        - NOTE: The passed iterable remains UNCHANGED

    - It is an object expression to be expanded in places where
        - 0 or more key-value pairs (for object literals) are expected

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Spread_syntax)


Syntax:
function <name>(<parameter>, ...<iterable>) {
    // function body
}

const <array> = [...<iterable>, <element_1>, ..., <element_n>];

const <object> = {
    ...<object>, 
    <key_1>: <value_1>,
    <key_n>: <value_n>,
};
